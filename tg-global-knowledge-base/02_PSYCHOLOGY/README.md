# 心理学セクション

このセクションでは、ビジネスに必要な心理学の知識を体系的にまとめています。

## 構成

### 01_Core_Concepts/
- **Cognitive_Biases.md** - 認知バイアス（確証バイアスなど）
- **Motivation_Theories.md** - 動機理論（マズローの欲求段階説など）

### 02_Business_Frameworks/
- **Consumer_Behavior.md** - 消費者行動
- **Organizational_Behavior.md** - 組織行動

## 学習の進め方

1. まず基本概念を理解する
2. ビジネスフレームワークを学ぶ
3. 実際のビジネスケースに適用する
