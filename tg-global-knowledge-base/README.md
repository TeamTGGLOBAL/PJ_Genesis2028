# TG Global Knowledge Base

ビジネスに必要な学際的な知識を体系的にまとめたナレッジベースです。

## 概要

このプロジェクトは、現代のビジネス環境で必要とされる多様な学問領域の知識を統合し、実践的なフレームワークとして提供することを目的としています。

## 構成

### 01_ECONOMICS/ - 経済学
- ミクロ・マクロ経済学の基本概念
- ゲーム理論、価格戦略などのビジネスフレームワーク

### 02_PSYCHOLOGY/ - 心理学
- 認知バイアス、動機理論などの基本概念
- 消費者行動、組織行動などのビジネス応用

### 03_SOCIOLOGY/ - 社会学
- ソーシャルネットワーク、組織文化などの基本概念
- チームダイナミクス、異文化コミュニケーションなどのビジネス応用

## 使用方法

1. 各セクションのREADMEを確認して概要を把握する
2. 基本概念から学習を開始する
3. ビジネスフレームワークを学び、実際のケースに適用する
